# Publicar desde el navegador, sin ejecutar archivos

1. Extrae el contenido de `Diario-de-Kiwi-Sin-Ejecutables.zip`.
2. Abre https://github.com/nmedvedy/kiwi.
3. Pulsa **Add file → Upload files**.
4. Arrastra al navegador todo el contenido de la carpeta extraída. Incluye la carpeta `.github`.
5. Escribe `Publicar Diario de Kiwi` como mensaje y pulsa **Commit changes**.
6. Abre https://github.com/nmedvedy/kiwi/actions y espera a que `Publicar Diario de Kiwi` termine en verde.
7. La web estará disponible en https://nmedvedy.github.io/kiwi/.

Si el proceso indica que Pages no está habilitado, abre **Settings → Pages** y elige **GitHub Actions** en **Source**. Después vuelve a **Actions** y pulsa **Run workflow**.
